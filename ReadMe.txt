Внимание! Это учебный сайт.
Практика: Сешаный код - нативный JS и первые уроки по ReactJS;

На нём два варианта запуска кода выполняющего одни и те же задачи:
  1. На реакте:
     - динамический вывод карточек
     - поле ввода, изменяющего заголовок
     - кнопка скрывающая карточки
     - динамическое выделение(отмена выделения) рамки для каждой карточки.

 2. На нативном JS:
     - динамический вывод карточек
     //- поле ввода, изменяющего заголовок
     //- кнопка скрывающая карточки
     - динамическое выделение(отмена выделения) рамки для каждой карточки.

     =======================================================

Чтобы  карточки, выполненные на Реакт-приложении, вывелись на сайте надо, помимо всех необходимых подключений в шапке сайта:

 1. установить Реакт-приложение через консоль-броузера
 (при запуске сайта через index.html нажми на F12 там будет преДЛОжение перейти по ссылке, следуй и устанавливай по-инструкции.
  Увидишь иконку реакта рядом с адресной строкой в броузере, значит приложение уже установилось!)
 2. Потом, запусти в корне проекта http-server.
      - установи его через команду                   npm install -g http-server
      - потом открой новую консоль и введи команду   http-server
      - всё сервер запущен!
      - теперь, в броузере вводи 127.0.0.1:8080
      (последняя цифра может меняться см.в консоли при подключении к серверу)

Всё! сайт собран!
